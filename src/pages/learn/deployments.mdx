## But what about multichain? {{ tag: "Multichain" }}

The ENS Protocol can be used on/for any chain! If you are building a dApp and want to use ENS you might have to add a [Mainnet RPC to your Wagmi config](/web/libraries).
This allows tools like wagmi to access mainnet, and load ENS information!

To enable this, simply specify `chainId: 1` in your config like so:

```tsx focus=6
import { useAccount, useEnsName, useEnsAvatar } from 'wagmi';

const Name = () => {
    const { data: ensName } = useEnsAddress({
        address: 'luc.eth',
        chainId: 1, // (1 = Ethereum, 4 = Goerli)
    });

    return <div>{ensName || address}</div>;
};
```
